# YouTube Stream Metadata

## Title
Monmouth Beach Live Cam — Shrewsbury River, NJ | Boating, Fishing, Weather & Tides (24/7)

## Description

Welcome to the Monmouth Beach live cam overlooking the Shrewsbury River in New Jersey. This 24/7 stream shows real-time river conditions, boating activity, fishing conditions, weather, and tides — all updated continuously with live Tempest weather data.

Great for:
• Checking boating and fishing conditions
• Seeing wind and tide before heading out
• Watching Shrewsbury River activity
• Monitoring Monmouth Beach weather in real time
• Enjoying a scenic NJ waterfront livestream

Live on-screen data includes:
• Wind speed & direction (ideal for boating & paddling)
• Temperature & humidity
• Rain status & storm indicators
• Barometric trends
• Tide stage & short-term forecast overlays

Weather data provided by Tempest Weather System, refreshed every few seconds.

🔧 Tech Behind the Stream (For Makers & Developers)

This stream is built using:

• VistterStream — an open-source Raspberry Pi live-streaming engine
• TempestWeather — a dynamic overlay generator pulling real-time Tempest API data

Source Code:
• https://github.com/nickdnj/vistterstream
• https://github.com/nickdnj/tempestweather
